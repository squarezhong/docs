#### Setup and Hold-time Violation

- Setup-time margin > 0

  $t_{clk} - t_{ffpd}(max) - t_{comb}(max) - t_{setup}$ 

  (ffpd: flip-flop propagation delay)

- Hold-time margin > 0

  $t_{ffpd}(min) + t_{comb}(min) - t_{hold}$

    2. Synchronous reset: State S~M-1~ -> S~0~
